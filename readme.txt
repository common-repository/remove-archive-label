Removes the “Category:”, “Tag:”, “Author:”, “Archives:” and “Taxonomy:” in the archive title.

== Description ==

Removes the “Category:”, “Tag:”, “Author:”, “Archives:” and “Taxonomy:” in the archive title.

No configuration is required


= Support =

Need help? Use the [support](https://wordpress.org/support/plugin/remove-archive-label)

== Installation ==

1. Upload `remove-archive-label.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Done!
